# 凌霄宫接入计划（2026-04-10）

## 目标
- 为新宗门“凌霄宫”接入自动化能力。
- 保持对现有 `xinggong` / `zongmen` / `yuanying` 等插件的最小影响。
- 先明确玩法边界，再决定是新插件还是复用现有配置结构。

## 当前代码现状
- `xiuxian_bot/plugins/zongmen.py`
  - 仅实现通用宗门日常：`点卯 + 传功`。
  - 没有“宗门类型/宗门专属玩法”抽象。
- `xiuxian_bot/plugins/xinggong.py`
  - 属于“星宫专属玩法插件”，包含观星台、启阵、深度闭关联动等完整逻辑。
- 结论：
  - “凌霄宫”从玩法复杂度上更像 `xinggong.py` 这种**独立宗门玩法插件**。
  - 不建议把这套逻辑直接塞进 `zongmen.py`，否则会把通用宗门日常和专属玩法耦合在一起。

## 截图里已知的凌霄宫玩法
- 状态：`.天阶状态`
- 日常：`.问心台`
- 核心登阶：`.登天路`
- 一天一层加速：`.引九天罡风`（1天后解锁）
- 短时加成：`.借天门势`（3周天解锁）
- 长老晋升：`.晋升凌霄长老`
- 太上长老晋升：`.晋升凌霄太上长老`

## 初步建议（最小可行）
1. 新增独立插件，例如 `lingxiaogong.py`。
2. 通用宗门日常 `zongmen.py` 保持不动。
3. 先做“可稳定自动化”的主链路，再逐步补充复杂分支：
   - 查询状态
   - 问心台
   - 登天路
   - 引九天罡风
   - 借天门势
   - 长老/太上长老晋升（如有明确判定条件）
4. 新增网页配置区块：
   - 开关
   - 轮询/尝试间隔
   - 是否自动问心台
   - 是否自动引九天罡风
   - 是否自动借天门势
   - 是否自动尝试晋升

## 已确认范围
1. 首期只处理：
   - `.问心台`
   - `.天阶状态`
   - `.登天阶`
2. 暂不处理：
   - `.晋升凌霄长老`
   - `.晋升凌霄太上长老`
3. `.引九天罡风` / `.借天门势` 先预留结构，不做自动化执行。
4. 实现形态：新增独立插件 `lingxiaogong.py`。
5. 与现有 `zongmen.py` 并行启用。
6. 默认假设：一个账号同一时间只会属于一个宗门。

## 已收到的真实回包

### `.天阶状态`
- 状态样例已确认，字段包括：
  - 当前进度：`4 / 12 阶`
  - 已完成周天：`0 轮`
  - 罡风淬体：`0/12` 或 `1/12`
  - 下一目标：如 `第 5 阶`
  - 下轮奖阶：如 `云门初启`
  - 登阶冷却：如 `3分34秒` / `3小时59分钟47秒`
  - 预计消耗：如 `238 点修为` / `280 点修为`
  - 当前成功率：如 `63%` / `56%`
  - 问心状态：
    - `今日已问心，但道印已在登阶中耗尽。`
  - 未解锁神通提示：
    - `引九天罡风: 未解锁（需完成 1 轮周天）`
    - `借天门势: 未解锁（需完成 3 轮周天）`

### `.问心台`
- 成功样例：
  - `凝出一道【澄明】之印` → 下次登天阶时，成功率显著提升。
  - `凝出一道【守一】之印` → 下次登天阶遭遇心魔失手时，可强行稳住道心。
  - 两种样例均获得 `20 点宗门贡献`。
- 已有道印样例：
  - `你今日已在问心台前留下一缕道印：【澄明】。下次登天阶时，成功率显著提升。`
- 今日已完成样例：
  - `你今日已在问心台前静坐过一次，道台不会再响应你。`

### `.登天阶`
- 成功样例：
  - `你消耗了 136 点修为，踏上了第 2 阶云阶。`
  - 奖励样例：`本次获得 200 点修为、80 点宗门贡献`、`额外收获：【金精】x1`
  - 状态更新样例：`当前云阶进度: 2 / 12，罡风淬体: 0 / 12`
- 冷却样例：
  - `九天罡风尚未再聚，请在 1小时14分10秒 后再试。`
  - `九天罡风尚未再聚，请在 45秒 后再试。`
- 失败样例：
  - `你消耗了 238 点修为，踏上了第 5 阶云阶。`
  - `你在阶前生出杂念，心魔趁虚而入，额外损失了 73 点修为。`
  - `当前云阶进度仍为 4 / 12，罡风淬体: 1 / 12`

## 最终确认
1. 实际命令名：`.登天阶`
2. 自动策略：
   - 冷却一到就自动尝试 `.登天阶`
   - 启动后先看 `.天阶状态`，再决定是否 `.问心台` / `.登天阶`
3. 网页配置：需要，首期加入最小项：
   - 总开关
   - 自动问心台开关
   - 自动登天阶开关
   - 状态轮询/尝试间隔
4. 已确认 `.天阶状态` 在持有道印时会直接显示，例如：
   - `问心状态：【澄明】 - 下次登天阶时，成功率显著提升。`
